## Description
An amazing Bullet 3D Physics Racing Videogame.

## Controls

    - KEYS for movement
    
    - R for reset de car

    - T for return to the last checkpoint 

### Debug
   
    - F1 show/hide collisions
   
    - F2 increase the mass
	
    - F3 set initial mass
   
    - F6 desactivate physics
   
    - F7 activate physics
